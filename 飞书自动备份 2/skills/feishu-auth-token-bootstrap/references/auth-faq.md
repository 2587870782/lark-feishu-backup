# 授权常见问题

## 授权页报错 20027
- 现象：用户授权时提示应用未开通相关权限。
- 原因：授权链接里的 `scope` 包含后台未申请权限。
- 处理：在飞书后台补齐权限后重新授权。

## 回调 URL 没有 code
- 现象：脚本提示回调 URL 中未找到 `code`。
- 原因：用户拒绝授权，或粘贴了错误 URL。
- 处理：重新授权，确保复制浏览器最终地址。

## 刷新 token 失败
- 现象：`code/main.py` 报 refresh token 已失效。
- 原因：`refresh_token` 过期、被轮换、文件损坏。
- 处理：重新执行 `code/get_initial_refresh_token.py` 生成并覆盖。

## token_store.json 缺失或损坏
- 现象：脚本报文件不存在或 JSON 解析失败。
- 处理：重新走一次授权流程让脚本自动重建文件。

## 安全注意事项
- 不在日志或截图中暴露 token 全值。
- 不把 `APP_SECRET` 提交到公共仓库。

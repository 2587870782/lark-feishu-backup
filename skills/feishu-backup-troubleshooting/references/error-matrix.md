# 错误码矩阵

## 授权与 token
- `20026/20037/20064/20073/20074`
  - 含义：refresh_token 无效或失效。
  - 动作：重新运行 `code/get_initial_refresh_token.py`。

## 导出与权限
- `1069902`（no permission）
  - 含义：当前 user/app 对目标文档无读取或导出权限。
  - 动作：检查 scope + 文档分享权限。

- `1069904`（invalid param）
  - 含义：导出参数不合法，常见于类型与扩展名不匹配。
  - 动作：核对 `type` 与 `file_extension`。

- `1069918`（file extension and type mismatch）
  - 含义：导出格式与文档类型不匹配。
  - 动作：使用脚本内映射规则，避免手工错配。

## 限频与稳定性
- `1069923` 或 HTTP `429`
  - 含义：触发限频。
  - 动作：提高重试、降低请求节奏、稍后重试。

- HTTP `500/502/503/504`
  - 含义：服务端或网关暂时不可用。
  - 动作：保留重试并观察是否可恢复。
